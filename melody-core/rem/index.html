# RemUI

Rem 的含义? “如果爱情有颜色，那一定是蓝色！”——致敬雷姆党！

最友好的react移动端组件库!

# 安装

建议react版本 >= v17.0.0; 支持react版本 >= v16.7.0;

yarn add @melody-core/rem
# or 
npm i @melody-core/rem --save

# 使用

import { Tag } from '@melody-core/rem';

const Demo = () => {
    const tagRef = useRef();
    return (
        <Tag size="large" ref={tagRef}>标签</Tag>
    )
}
